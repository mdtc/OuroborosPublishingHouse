Design

For Ouroboros Publishing House, I have decided to focus on a classic retro-design, I wanted to refer the users to the feeling of being reading an old Sci-Fi book. The combination of clean black and white, with retro sci-fi art, makes this website and business a very specialized kind of publishing house.

Despite the use of images as background, is not considered the best practice for website design. I decided to start the home page in this way. It is very well known the massive importance of art images, book covers and drawings in speculative fiction, so the first objective of this website was trying to simulate holding an old Sci-Fi book with retro art as cover. Click here if you want to know a little bit more of The history of Art in Sci-Fi Book Covers I also have applied a hover with aplha when the section is focused, in this way text is not lost for the image.

Apart from the home page that is made from big image-background sections, I tried to keep every other page as clear and tidy as possible, specially the authors and writter pages, where a lot of information has to be read.

This website meets the criteria of the new web standards, its build with the markup language XHTML, which is more strict with the way the browsers read the tags. I also have used CSS3. I have avoided using Javascript for this particular website.

Mobile Design

The home page serves as a big graphic menu, ready to be scrolled down. Each section has a personal background which identifies with the topic is referring to. When a page is accessed, the navigation menu will display the same background image as the section on the home page. In this way, the user is visually linked to every different page of the website.

I have not used a drop-down menu because of the lack of javascript in the code, I have tried to make it intuitive and according to the design.

Browser Design

The website has been built with the mobile-first philosophy, that means the first design development was made for small devices. However, it is responsive and design changes in larger screens. The home page displays a navigation menu, the authors and books have been placed horizontally rather than vertical, and the images show a lot more detail in bigger screens.

Typography

As I commented in the code, I have struggled a lot, trying to link the fonts from an external source such as Google Fonts. Due to my decision to use XHTML instead of html5, the link provided by google did not work well. After extensive research, I found out that the problem was the use of the reserved character "ampersand" which had to be escaped. I have used three different fonts on the website:

  • Google font Allerta for H1 headers
  • Google font Cinzel for H2 headers
  • Helvetica for general content

Images

Some images used on this website might be protected by copyright. They belong to the original artist or creators; this website is only for academic purposes. I also have modified the size of the images making them "web-friendly" reducing the weight through photoshop.

Logo

The logo has been drawn on illustrator and saved in png format with alpha. It refers to the main subject in Sci-fi; Universe and a serpent eating its own tail as the ancient symbol of The Ouroboros.
